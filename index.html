Proyecto 1

En esta sección, puedes gestionar el inventario de nuestros productos. Completa el formulario a continuación para añadir nuevos productos al inventario. Asegúrate de ingresar todos los detalles necesarios, incluyendo el código del producto, el nombre, el costo, el porcentaje de venta y el precio de venta. También puedes cargar una imagen del producto para una mejor identificación visual. Una vez que hayas completado el formulario, haz clic en "Guardar" para registrar el producto.

Proyecto 2

Este proyecto está diseñado para proporcionar a los usuarios una experiencia fluida y agradable al interactuar con la editorial y acceder a información clave sobre los productos y servicios que ofrece Ediciones fares. El sitio está estructurado de manera intuitiva y elegante, facilitando el acceso a información relevante y la comunicación con la editorial.

Proyecto 4
